This is the publicworship.life marketing site, migrated into events-os from
[lilseyi/public-worship](https://github.com/lilseyi/public-worship). It's built
with [Astro](https://astro.build) and served as static assets by the
Cloudflare router worker in `infra/router` (canonical URL:
https://publicworship.life). Page copy and structured content live in
`src/content/*.yaml`.
